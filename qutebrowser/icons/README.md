qutebrowser icon bundle.

Files:
- `qutebrowser-*.png`: per-size app icons
- `qutebrowser.png`: generic png
- `qutebrowser.ico`: multi-size ico
- `qutebrowser.icns`: icon under `Resources/qutebrowser/icons/`
- `qutebrowser-root.icns`: root app icon at `Contents/Resources/qutebrowser.icns`
- `qute7-final.png`: cleaned source asset used for current icon set

Install on another machine:

```bash
~/dotfiles/qutebrowser/install-icons
```

Optional custom app path:

```bash
~/dotfiles/qutebrowser/install-icons "/Applications/qutebrowser.app"
```
