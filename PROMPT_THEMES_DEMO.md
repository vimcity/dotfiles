## Try Them Now

```bash
# Switch between themes instantly
theme-rp        # Rose Pine
theme-cat       # Catppuccin
theme-nord      # Nord
theme-dracula   # Dracula
theme-minimal   # Minimal
theme-gruvbox   # Gruvbox

# See all themes
theme-list
```

## What Changes

When you switch themes, these parts of your prompt update immediately:

- **Directory segment** - background and text color
- **Branch/Git segment** - colors for branch name and status
- **Virtual env indicator** - colors update
- **Prompt character** ($) - color changes
- **Right prompt** - duration, job count, error code colors
- **Git status icons** - added/modified/deleted indicator colors
